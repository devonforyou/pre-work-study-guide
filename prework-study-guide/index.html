HTML

  • The head element contains information about the webpage.
  • The head is not visible to users, it is used for the browser to structure the page properly.
  • The body element represents the visible content shown to the user.
  • They body contains child elements such as h1, li, ul, ol, p, img.
  • Br allows a empty space between blocks of content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes can be assigned to anything we want by a class selector, class="card" and .card.
  • Preceding is the selector [property: value; property: value:]. Declarations are property and value.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout -f branch-name: forces a change in branches.
  • git add -A stages our changes.
  • git commit -m allows us to add a message to our commit.
  • Make sure our current branch is up to date before opening a pull request w/ git pull origin main.
  • Make sure to update our local branch w/ git checkout main, then git pull origin main.
